MATLAB skills

In this lab you will observe one object in different wavelengths to get multiple images. Then you'll have to clean, calibrate, and combine all the maps. Here are some MATLAB image processing commands that you'll be using in the lab.
  • Flip and Rotate
  • Graphical Input
  • Shifting Images
  • Contour maps
You will also need to write your own MATLAB functions to help you find out how the intensity varies across the nebula along the major and minor axes. Here's a simple instruction about how to write a MATLAB function.
